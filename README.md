# E-Commerce SQL Database

## 📖 Description
This project is a relational SQL database designed for an E-Commerce Management System. It manages core online shopping operations such as customer information, products, categories, shopping carts, orders, payments, and inventory.

The project demonstrates SQL database design, normalization, primary and foreign key relationships, and data management.

## ✨ Features
- Customer Management
- Product Management
- Category Management
- Shopping Cart
- Order Management
- Payment Management
- Inventory Management
- Relational Database Design

## 🛠️ Technologies Used
- SQL
- MySQL

## 🚀 Getting Started

### Prerequisites
- MySQL Server
- MySQL Workbench (or any SQL client)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/E-Commerce-SQL-Database.git
   ```

2. Open MySQL Workbench.

3. Import the SQL file:
   ```
   e commerce full.sql
   ```

4. Execute the script to create the database and tables.
